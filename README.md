# ReadMe

PowerTools目的是扩展VSCode文本处理能力,提供转换大小写、base64编码解码、排序行，哈希文本等功能。 

## 主要功能

### 文本处理

- 大小写转换
- base64编码/解码
- 移除空行
- 移除首尾空格
- 筛选行

### 行处理
- 按文本排序行
- 按数字排序行
- 翻转行

### 其他功能
- 选中文本哈希
- 生成32位随机串
- 生成强密码
- 对选中文本求和
- 对选中文本取平均
